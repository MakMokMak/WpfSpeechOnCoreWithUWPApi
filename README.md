# WpfSpeechOnCoreWithUWPApi

SpeechSynthesizer クラスを利用する部分の説明は、[ブログの記事](http://www.makcraft.com/blog/meditation/2019/12/14/using-uwp-speechsynthesizer-in-wpf-on-net-core-3-0/)をご覧ください。
